TYCOON

During my internship at Liseberg- an amusement park, I was tasked with developing a web application to transform raw data into a fun and interactive experience. The goal was to move away from traditional tools like Excel and create a dynamic way for users to explore daily statistics. The result is an engaging web app that makes data visualization both intuitive and enjoyable.

Key Features

  • Real-Time Data Updates: The application receives data from Azure Service Bus, processes it, and updates the visualizations in real time.

  • Efficient Data Handling: The backend uses Redis Cache for fast data access and MS SQL for reliable storage, ensuring smooth performance even with large datasets.

  • Automated Infrastructure: The entire system is hosted on Azure, with infrastructure deployment automated using Bicep for consistency and scalability.

Benefits for Users

  • Interactive and Fun: Instead of sifting through spreadsheets, users can explore data in a visually appealing and interactive way.

  • Real-Time Insights: The application provides up-to-date statistics, helping users quickly see the amusement park activity

  • User-Friendly: The clean and modern design makes it easy for anyone to use, regardless of their technical expertise.

Technologies Used:

  • Frontend: Blazor WebApp

  • Backend: ASP. NET API, Azure Service Bus, Redis Cache, MS SQL

  • Infrastructure: Azure (hosting and services), Bicep (for automation)

Impact:

This project demonstrates my ability to create user-centric applications that simplify complex data and make it accessible to everyone. By combining modern technologies with a focus on design and usability, I’ve built a tool that turns mundane statistics into an engaging and interactive experience- perfect for bringing a touch of fun to data analysis.
